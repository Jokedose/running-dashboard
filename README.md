# 📊 Running Dashboard

Personal running dashboard for tracking 10K training — built with Vite + React, deployed on GitHub Pages.

🔗 **Live:** [jokedose.github.io/running-dashboard](https://jokedose.github.io/running-dashboard/)

---

## Pages

| Page | Content |
| --- | --- |
| **Today** | Daily readiness — Recovery, HRV, Sleep, Load ratio, session recommendation |
| **Calendar** | Training plan schedule + session status |
| **Activities** | Run logs — distance, pace, HR, Z2%, drift |
| **Race** | Race readiness score, 10K projection chart, strengths/risks |
| **Weekly** | Weekly summary — total km, session quality, planned vs actual |
| **Trends** | Long-term trend charts — pace, HR, Z2% |
| **Zone 2** | Zone 2 analysis — drift, decoupling, sweet spot |
| **Gear** | Shoe mileage tracker |
| **Plan** | Long-term training phases |

---

## Tech Stack

React · Vite · MUI · Recharts · Supabase · GitHub Pages
